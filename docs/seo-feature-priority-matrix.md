## 1. SEO FEATURE PRIORITY FRAMEWORK

### Scoring Formula

Score = (Search Volume × Intent Match × CTR Potential × Competition Factor) / Implementation Effort

Where:
- Search Volume: 1 (< 500/mo) → 5 (> 20,000/mo)
- Intent Match: 1 (low) → 5 (perfect match to product feature)
- CTR Potential: estimated organic CTR at current vs target ranking
- Competition Factor: 1 (very competitive) → 5 (low competition)
- Effort: 1 (very high) → 5 (trivial)

## 4. CONTENT-PRODUCT FLYWHEEL OPPORTUNITIES

These features generate SEO-useful content at scale:

### Flywheel 1: Tool Pages → Shareable Outputs
- IRL calculator result → shareable page with unique URL (e.g., /outils/calculateur-irl/result/2026-Q1-1100)
- Yield calculator → shareable investment summary
- Receipt generator → downloadable + shareable PDF (drives brand mentions)

### Flywheel 2: User-Generated Template Downloads → Backlinks
- Every template download from /outils/* can trigger an opt-in email
- Users who share templates on landlord forums = organic backlinks
- Track: template downloads per month, source attribution

### Flywheel 3: Glossary Expansion via Product Data
- Product generates real lease data: common clauses, typical charges, average deposits
- Glossary entries can reference real-world data ("Average deposit in Paris is X according to Rent-Ready data")
- Positions as authoritative data source

### Flywheel 4: Inspection Reports → Photo-Rich Content
- Inspection module (V2): users create état des lieux with photos
- Anonymized aggregated data can power: "What damages cost landlords most?" articles
- Creates unique content competitors cannot replicate

## 5. COMPETITIVE SEO GAP ANALYSIS

### Key Competitors Analyzed

| Competitor | Domain Authority | Monthly Traffic (est) | Strongest Content |
|---|---|---|---|
| Rentila | Medium | 80K/mo | bail templates, quittance tool |
| Serenity (Otera) | High | 150K/mo | legal guides, comparison pages |
| Bricks | Medium | 40K/mo | startup content, comparisons |
| ImmVision | Low | 20K/mo | software features |

### Gaps We Can Win

1. **Better tool pages**: Competitors have PDF downloads but no interactive calculators. We build interactive tools → higher engagement → better rankings.
2. **Fresher legal content**: Loi ELAN updates, 2026 IRL values — competitors lag 3-6 months. We update instantly.
3. **Structured data**: Competitors do not use FAQ schema or HowTo schema. We implement on all content pages.
4. **Video SEO**: No competitor has YouTube presence for "how to" landlord content. Opportunity: 5-10 quick tutorial videos.
5. **E-E-A-T signals**: Add author bios (legal expert), cite law sources, add publication dates + "last reviewed" dates.

### Content Types Competitors Rank With
- Long-form legal guides (1,500-3,000 words): YES → we copy
- Template download pages: YES → we copy + improve with online editor
- Comparison pages: YES (weak) → we do better
- Calculator tools: NO → first-mover advantage
- Video content: NO → opportunity

## 7. PRE-LAUNCH SEO CHECKLIST

Every feature/page must pass this before going live:

### Technical
- [ ] Page renders in < 2.5s LCP on mobile (PageSpeed Insights)
- [ ] CLS < 0.1 across all viewport sizes
- [ ] robots.txt allows crawling of this path
- [ ] Page included in sitemap.xml
- [ ] Canonical URL set (<link rel="canonical" href="...">)
- [ ] No broken internal links on page
- [ ] HTTPS enforced (no mixed content)

### On-Page SEO
- [ ] Unique meta title (50-60 chars) containing primary keyword
- [ ] Unique meta description (140-160 chars) benefit-focused
- [ ] H1 contains primary keyword (exactly once per page)
- [ ] H2/H3 structure logical (no heading level skips)
- [ ] Primary keyword in first 100 words of body content
- [ ] Images have descriptive alt text
- [ ] Word count appropriate for intent (> 800 for info, > 400 for transactional tools)

### Structured Data
- [ ] JSON-LD schema markup added (Article / FAQ / HowTo / Product as appropriate)
- [ ] Validated via Rich Results Test (no errors)
- [ ] BreadcrumbList schema for all non-homepage pages

### Linking
- [ ] Page links to at least 2 internal related pages
- [ ] At least 2 existing pages have been updated to link to this new page
- [ ] External sources cited (for factual claims, link to law.fr or legifrance.gouv.fr)

### Content Quality (E-E-A-T)
- [ ] Legal accuracy verified (landlord law in France)
- [ ] French spelling and grammar checked (no UK/US English)
- [ ] Author or "reviewed by" attribution visible for legal/advice content
- [ ] Date published and last updated shown on article pages
- [ ] No AI writing tells (no "Certainly!", no em-dash overuse, natural French prose)
