## Strategy

Each cluster targets a distinct topical domain in French rental property management.
Clusters are built hub-and-spoke: a pillar page (hub) with supporting articles (spokes)
that all link back to the pillar and to each other, spreading link equity and
demonstrating comprehensive topical coverage to Google.

**5 core clusters covering 100+ keyword terms:**

## Topical Authority Tracking

Google measures topical authority via:
1. **Coverage:** number of related keywords ranked for
2. **Depth:** average position for cluster keywords
3. **Freshness:** content update frequency
4. **Crawl frequency:** how often Googlebot revisits cluster pages

**Target:** Move from "thin" to "comprehensive" coverage within 6 months.
Measure via: Google Search Console → Coverage report → Avg. position for cluster keywords
