## FAQ

**Q: Quels régimes fiscaux sont supportés ?**
A: RentReady supporte le régime micro-foncier et le régime réel. Les exports sont formatés différemment selon le régime : pour le micro-foncier, seuls les revenus totaux sont nécessaires ; pour le réel, les charges catégorisées sont exportées avec le détail.

**Q: Puis-je exporter uniquement les transactions d'une période donnée ?**
A: Oui. Vous pouvez filtrer par date : année civile, année fiscale, ou période personnalisée. L'export inclura uniquement les transactions de la période sélectionnée.

**Q: Les données sont-elles conservées d'une année sur l'autre ?**
A: Absolument. Toutes vos transactions et données comptables sont conservées sans limite de temps. Vous pouvez accéder à vos exports comptables de 2022, 2023, 2024 et plus — utile en cas de contrôle fiscal.

**Q: Les travaux déductibles sont-ils inclus dans l'export ?**
A: Oui. Si vous avez enregistré des dépenses de travaux dans RentReady, elles sont incluses dans l'export réel avec la catégorie appropriée. Vous pouvez également saisir des charges de travaux non enregistrées via la comptabilité.
