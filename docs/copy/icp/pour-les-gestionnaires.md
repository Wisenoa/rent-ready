## ICP-Specific FAQ

**Q: Quelle est la différence entre l'offre PRO et l'offre STARTER ?**
A: PRO est conçu pour les property managers et gestionnaires de portefeuille. Les différences clés : nombre de lots illimités (vs 10 en STARTER), portail owner en marque blanche, API ouverte, reporting consolidé de portefeuille, workflow équipe avancé, et support prioritaire.

**Q: Pouvez-vous gérer des biens dans plusieurs pays (France, Belgique, etc.) ?**
A: Actuellement, RentReady est optimisé pour la législation française. Les fonctionnalités comme les quittances, les modèles de bail, et les calculs IRL sont conformes au droit français. Pour des biens à l'étranger, contactez-nous pour discuter de votre cas.

**Q: Y a-t-il des frais d'installation ou de setup ?**
A: Non. L'inscription est自助. Un onboarding guidé est inclus pour vous aider à configurer votre espace, importer vos biens existants, et invites vos owners. Un support technique dédié est disponible pendant les 30 premiers jours.

**Q: Quelle est la politique de migration des données ?**
A: Vos données vous appartiennent — toujours. Vous pouvez exporter l'intégralité de vos données à tout moment en CSV/Excel. En cas de départ, un export complet est disponible. Aucune donnée n'est retenue.
