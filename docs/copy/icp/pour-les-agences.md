## ICP-Specific FAQ

**Q: Combien de lots/gérance pouvez-vous supporter ?**
A: RentReady STARTER couvre jusqu'à 50 lots. L'offre PRO est recommandée pour les agences de 50 à 200 lots. Pour les portfolios plus importants, contactez-nous pour une offre sur mesure.

**Q: Pouvez-vous vous interfacer avec notre logiciels de comptabilité existant ?**
A: L'export CSV/Excel de RentReady est compatible avec la plupart des logiciels comptables (Cegid, Sage, Pennylane, etc.). Pour une intégration API plus poussée, c'est disponible en offre PRO ou sur demande.

**Q: Le portail propriétaire est-il en marque blanche ?**
A: Le portail propriétaire utilise le branding RentReady en standard. En offre PRO, le portail peut être personnalisé avec le nom et le logo de votre agence.

**Q: Quelles sont les obligations réglementaires pour une agence utilisant un outil SaaS ?**
A: Les données de vos clients sont hébergées sur des serveurs sécurisés en France, conformes RGPD. Nous ne revendons aucune donnée. Un DPA (Data Processing Agreement) est disponible sur demande pour vos obligations de confidentialité.
