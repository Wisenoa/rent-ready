## SECTION 2: CONTENT BRIEF TEMPLATE

For each planned article, writers should use this brief structure:

### Article Brief Template
```
## Metadonnees
- **Slug:** [url-slug]
- **Cluster:** [1-5]
- **Type:** Blog / Pillar / Guide
- **Word count:** [target]
- **Target keyword (primary):** [keyword]
- **Target keywords (secondary):** [2-3 variants]

## Search Intent
[Informational / Commercial / Transactional]
[What the searcher wants to know or do]

## Structure (H2/H3 Outline)
### H2: [Section title]
#### H3: [subsection]
#### H3: [subsection]
### H2: [Section title]
...

## Required Sections
1. **Intro** (100-150 words): Hook + what this article covers + why it matters
2. **Body** (main content): Answer the query comprehensively
3. **FAQ** (3-5 Q&A): Based on People Also Ask results for target keyword
4. **CTA** (conclusion): Soft product mention + link to relevant tool/page

## Internal Link Targets
- [Page 1] -- anchor: "[descriptive text]"
- [Page 2] -- anchor: "[descriptive text]"

## Competitor Reference
[URL of top-ranking article for target keyword]

## E-E-A-T Requirements
- Author: RentReady editorial team
- Include specific legal references (loi, article, decreet)
- Cite official sources (service-public.fr, legifrance.gouv.fr)
- Publication date prominently displayed
- Update timestamp shown

## Checklist Before Publishing
[ ] Primary keyword in H1, first paragraph, and at least 2 subheadings
[ ] Meta title 60 chars, includes primary keyword
[ ] Meta description 155 chars, includes CTA signal
[ ] Internal links to 2+ existing pages
[ ] External links to official sources (service-public.fr, etc.)
[ ] FAQ section with FAQPage JSON-LD schema
[ ] Author schema and Article schema present
[ ] Images have descriptive alt text
[ ] Readability score 70+ (Hemingway or similar)
```
